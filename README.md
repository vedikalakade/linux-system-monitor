# Linux System Monitor

A Bash scripting project that displays important system information and server health details on Linux systems.

## Features

* Displays current user information
* Shows date and time
* Displays operating system details
* Shows CPU information
* Displays memory usage
* Displays disk usage

## Technologies Used

* Linux
* Bash Scripting
* Shell Commands

## Project Structure

```text
linux-system-monitor
├── README.md
├── myscript.sh
└── screenshots
```

## How to Run

```bash
chmod +x myscript.sh
./myscript.sh
```

## Sample Output

The script displays:

* User Name
* Current Date & Time
* Operating System
* CPU Details
* Memory Usage
* Disk Usage
